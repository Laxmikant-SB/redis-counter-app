# Redis Counter App

![CI Pipeline](https://github.com/Laxmikant-SB/redis-counter-app/actions/workflows/ci.yml/badge.svg)

A simple Python app demonstrating CI/CD with Docker and GitHub Actions.

## What it does
Connects to Redis and increments a visit counter every 2 seconds.

## Tech Stack
- Python 3.11
- Redis
- Docker
- GitHub Actions CI/CD
